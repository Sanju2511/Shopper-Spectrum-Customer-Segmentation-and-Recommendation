# Shopper Spectrum: Customer Segmentation and Product Recommendation

## 📌 Project Overview
This project analyzes e-commerce transaction data to understand customer purchasing
behavior, segment customers using RFM analysis and clustering techniques, and build
a product recommendation system using item-based collaborative filtering.

The project combines Exploratory Data Analysis (EDA), unsupervised machine learning,
and a deployable Streamlit application to demonstrate both analytical and practical
business insights.

---

## 🎯 Objectives
- Perform detailed Exploratory Data Analysis (EDA) on online retail data
- Segment customers using Recency, Frequency, and Monetary (RFM) analysis
- Apply clustering algorithms to identify meaningful customer segments
- Build a product recommendation system using collaborative filtering
- Deploy an interactive Streamlit application for real-time predictions

---

## 🧠 Techniques Used
- Data Cleaning and Feature Engineering
- Exploratory Data Analysis (EDA)
- RFM Analysis
- KMeans Clustering
- Elbow Method and Silhouette Score
- Item-based Collaborative Filtering
- Cosine Similarity
- Streamlit for deployment
