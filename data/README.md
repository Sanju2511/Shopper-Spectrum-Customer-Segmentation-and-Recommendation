# Data Setup

This project expects the raw dataset at:

`data/raw/online_retail.csv`

## Steps
1. Obtain the Online Retail transaction dataset.
2. Place it at `data/raw/online_retail.csv`.
3. Run the notebook or app from repository root.

## Notes
- Raw data is excluded from git via `.gitignore`.
- Keep filename exactly `online_retail.csv` for default paths.
